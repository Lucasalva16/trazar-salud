# Trazar - Data Pipeline

Este repositorio documenta el pipeline de datos utilizado para analizar
las prácticas médicas y módulos de clínicas en el proyecto Trazar.

## Herramientas

- Power BI
- Power Query
- SharePoint
- Excel

## Fuentes de datos

Datos operacionales provenientes del sistema SII y CUP exportados a Excel y
almacenados en SharePoint.

## Pipeline

1. Ingesta de archivos mensuales
2. Limpieza y normalización
3. Clasificación de prácticas
4. Modelado en esquema estrella

## Output

Modelo analítico utilizado en dashboards de Power BI.
